# 安装

Note

从6.0.0开始，Kibana 只支持64位操作系统。

Kibana 提供以下格式的安装包：

tar.gz/zip tar.gz 包用来在 Linux 和 Darwin 系统下安装，也是最方便的一种选择。
zip 包是唯一支持 Windows 系统的安装包。

使用 .tar.gz 安装 Kibana 或者 在 Windows 上安装 Kibana
deb deb 包用来在 Debian、Ubuntu 和其他基于 Debian 的系统下安装，Debian 包可以从 Elastic 官网或者我们的 Debian 仓库中下载。

使用 Debian 包安装 Kibana
rpm rpm 包用来在 Red Hat、Centos、SLES、OpenSuSe 以及其他基于 RPM 的系统下安装。RPM 包可以从 Elastic 官网或者我们的 RPM 仓库下载。

使用 RPM 包安装 Kibana
docker Elastic Docker 仓库中有现有的可以运行 Kibana 的 Docker 镜像。

Docker 容器中运行 Kibana
brew macOS 上可以通过 Homebrew 安装 Kibana。

macOS 上通过 Homebrew 安装 Kibana

如果您安装的 Elasticsearch 是受 X-Pack 安全插件保护的，请查看X-Pack安全插件获取更多的配置信息 。
